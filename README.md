# ajax Api Example 
- 서버 실행 
``` 
npm start
```
- 디렉토리 구조

``` 
public
    |____ajaxAwaitAsync.html  -- Await/Async 예제 파일
    |____ajaxCallback.html    -- ajax (Callback 패턴으로) 적용한 예제 파일
    |____ajaxStep1.html       -- ajax (local api) 적용한 예제 파일
    |____ajaxStep2.html       -- ajax (external api) 적용한 예제 파일
    |____ajaxStep3.html       -- ajax (cross domain) 적용한 예제 파일
    |____ajaxStep4.html       -- ajax (promise pattern) 적용한 예제 파일
    |____ajaxSync.html        -- ajax (동기적으로) 적용한 예제 파일
    |____exercise.html        -- 실습 파일
    |____viewHtml.html        -- 결과화면
css
    |____page.css             -- 스타일
    |____reset.css            -- 리셋 스타일
js
    |____api.js               -- step1, step2에 사용한 js 파일
    |____handlebar.js         -- handlebar 라이브러리
    |____menu.json            -- handlebar 라이브러리
    |____step1.js             -- step1에 사용한 js 파일
    |____step2.js             -- step2 에 사용한 js 파일             
    |____step3.js             -- step3 에 사용한 js 파일  
    |____step4.js             -- step4 에 사용한 js 파일  
```
